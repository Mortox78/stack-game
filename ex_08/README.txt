Description :
Stack is game where you will have to stack blocks in order to create a tower

How to build it:
Clone it
Go in the folder "OpenGL/GP1_NOVO_S" and then write in the terminal : "make"

How to launch the game:
Go in the folder "OpenGL/GP1_NOVO_S" and then write in the terminal : "./gl"


Technical uses:
OpenGL
C++

Features:
Score
Press "space" in order to stack block on the tower
If you've lost, press "space" if you want to start an other game
If you've lost, you will the entire tower you've built turn on itself



There are no lights and no transitions betweens "game over" and the "restart"
